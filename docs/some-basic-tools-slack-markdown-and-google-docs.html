Data Science for the Liberal Arts

2 Some basic tools: Slack, Markdown, and Google Docs

On Markdown, see (Freeman and Ross (2017), Chapter 3). Some Markdown editors include Typora, Atom, and Ghostwriter. There is a simple markdown editor in Slack (for ‘posts’).

For beginning with R markdown, consider Carmichael (2017) Getting started as well as the first chapter of Wickham and Grolemund (2016). Install R and R studio on your own laptop. If you get stuck, reach out to others on Slack; if you don’t get stuck, help your classmates.
